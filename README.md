🌿 EcoRoute AI – Intelligent Eco-Friendly Route Planner

EcoRoute AI is a smart route recommendation system that uses machine learning models to predict CO₂ emissions and suggest the most environmentally friendly path instead of just the shortest one.

🚀 Features
🌍 Eco-friendly route selection based on CO₂ emissions
🤖 AI Models:
Decision Tree (High Accuracy)
Linear Regression (Fast Prediction)
📊 Dynamic route comparison table
➕ Add multiple route options
🏆 Eco-Win indicator for greener choices
⚡ Fast and interactive UI
🛠️ Tech Stack


Frontend:

HTML, CSS, JavaScript

Backend:

FastAPI (Python)

Concepts Used:
Machine Learning (Prediction Models)
Event-driven Programming
Dynamic UI Rendering
REST API Integration
📂 Project Structure

⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/your-username/ecoroute-ai.git
cd ecoroute-ai
2️⃣ Install Dependencies
pip install fastapi uvicorn
3️⃣ Run Backend Server
uvicorn main:app --reload
4️⃣ Open in Browser
http://127.0.0.1:8000

🧠 How It Works
User inputs multiple route options
Selects vehicle type and AI model
Backend calculates CO₂ emissions based on:
Distance
Traffic conditions
Road type
AI model predicts emissions
System recommends the most eco-friendly route
